Project structure
=================

The implementation of the *Job Monitor* project is structured in several
modules, which implement the following components:

- *Job Manager* entity
- *Job Monitor* entity
- Database-related functionality
- Data models related to tasks and analytics
- Logging facility
- Miscellaneous helpers
